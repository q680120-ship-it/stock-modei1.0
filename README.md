# stock-modei1.0
價格突破：收盤價必須創下 20 日最高點（確保短線動能轉強）。  趨勢過濾：股價必須站在 200 日年線 之上（確保不是在空頭市場裡接刀）。  成交量爆發：當日成交量須大於過去 5 日均量的 1.5 倍（確保有主力資金進場）。  強勢但不追高：RSI 需大於 50（多頭佔優），但小於 80（避免短線極端過熱）。移動停利線 (Trailing Stop)：根據進場後的最高點，自動向下移動 10% 繪製出「橘色防守線」。  動態預警機制：  出場警示：當股價收盤跌破橘色線，系統會彈出紅色錯誤提示，通知你獲利了結。  過熱警示：當 RSI 超過 80，系統會彈出黃色警告，建議你分批減碼。
